ASSET PLACEMENT INSTRUCTIONS

For the frontend to look correct, please place the following files in this folder (frontend/public/assets/):

hero.jpg

Recommended Size: 1920x1080px

Description: High quality background image for the home page.

Format: JPG or WEBP

logo.glb

Description: 3D Model of your logo.

Format: GLB (Compressed GLTF).

Optimization: Keep it low poly (under 1MB recommended) for web performance.

Tool hint: Use https://gltf.report/ to compress models.

placeholder.jpg

Recommended Size: 500x500px

Description: Fallback image for products that don't have an image link.

NOTE: You can change these filenames in frontend/src/config.js if you prefer different names.
